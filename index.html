Cellavie PyPI


This is the private Python package index of Cellavie for the ReGen suite of software packages.
See the documentation on Cellavie to find out how to use this index. In a nutshell, provide this index to pip when installing a package: 

pip install <package_name> --extra-index-url https://cellaviebe.github.io/regen-pypi/
Be aware that the packages listed here are all private. You will need to be able to authenticate with the server in order to install them.
